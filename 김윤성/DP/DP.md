# 다이나믹 프로그래밍 (DP)

- 동적계획 알고리즘이라고도 한다.

### 키워드
- 문제 해결
- 점화식
- 항 밑에서부터 차례로

### 주요 특징
- 최적화 문제를 해결하는 알고리즘

- 가장 작은 모든 부분 문제들의 해를 저장해 놓는다.
- 상위 문제들의 해를 구하기 위해 이것들을 활용한다.

- 작은 모든 부분 문제들의 가능성을 고려해서 다음 부분 문제의 최적해를 결정한다.
- 따라서 항상 최적의 결과를 얻는다.

### 주요 문제
ex) 피보나치
재귀적으로 하면 중복된 연산
DP로 해결하면 배열로 해결 가능
중간결과를 저장 하는지 안하는지에 따라 극적인 차이 발생

### DP 문제 해결 과정

1. 테이블 정의하기
2. 점화식 찾기
3. 초기값 정하기

- 수학적 직관력을 갖고 있지 않는 이상 점화식을 찾기 어려움
- 주어진 문제가 DP라는것 자체가 찾기 힘듬
